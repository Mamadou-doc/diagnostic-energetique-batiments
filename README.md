# Diagnostic énergétique des bâtiments

Ce projet analyse les performances énergétiques des bâtiments (DPE) à partir de données réelles.

## Objectifs
- Identifier les passoires thermiques
- Analyser les facteurs influençant la consommation
- Visualiser les profils énergétiques
- Proposer un diagnostic prédictif

## Application interactive
Une application Streamlit permet de :
- visualiser les tendances globales
- analyser les passoires thermiques
- explorer les profils via ACP
- simuler un diagnostic énergétique

## Lancer le projet

```bash
streamlit run app.py
